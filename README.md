# 📹 Clipando Cast — AI Video Clip Generator

Um gerador inteligente de clipes automáticos a partir de vídeos longos, com transformações no formato vertical (9:16) para plataformas como **YouTube Shorts, Instagram Reels e TikTok**.

Esse projeto usa **IA para detectar partes relevantes do vídeo**, transcrever áudio e gerar clipes com estilo profissional.

---

## 🚀 Funcionalidades Principais

- 🎯 Detecção de trechos relevantes automaticamente
- 🗣️ Transcrição de voz usando **Whisper**
- 🤖 Processamento de texto com **Google Gemini**
- 📱 Conversão para formato vertical 9:16
- ✨ Estilização com fundo desfocado + legendas embutidas
- 🎛️ Modo de revisão interativo
- ⚡ Suporte à geração de múltiplos clipes por vídeo
